# FamilyFrame （家庭相框）

## How it works

用户关注一个微信订阅号，往该订阅号发送图片;发送成功之后，图片便直接同步到《家庭相框》APP。

## Why it works

主要是使用了Firebase的实时推送功能。用户往订阅号发送图片的时候，微信后台会生成一个图片URL

到开发者后台，开发者后台只需要把该链接添加了Firebase的目标地址即可。而在客户端，则会收到

Firebase的推送，从而可以实时自动更新图片。
